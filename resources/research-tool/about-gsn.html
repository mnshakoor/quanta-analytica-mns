Overview

This single file dashboard polls public RSS and Atom feeds, normalizes items into events, and renders three synchronized views. It runs in a browser with no backend. Feeds are cached in localStorage for 15 minutes for speed.

Tip: use a small local web server in dev. Example python3 -m http.server 8000 then open http://localhost:8000.

Layout

Header

  • Live status and event count
  • Last update time
  • Refresh and Theme buttons

Left sidebar

  • Search box for instant keyword filter
  • Filters for Date range, Event types, Sources, Region
  • Top 5 Countries widget

Main column

  1. KPI dashboard
  2. Satellite map with heatmap and markers
  3. Event network visualization

How to use

Search and filter

  • Type keywords such as cholera, ransomware, earthquake, or a country name
  • Set a date window to 7, 30, or 90 days
  • Toggle event types to focus the view
  • Uncheck noisy sources
  • Pick a region to scope analysis

Map

  • Heatmap shows density. Circle markers show events by type color
  • Click a marker to see details and the source link
  • Use the JSON or CSV buttons for export from the current filtered set

Network

  • Nodes are events. Links show similarity by type, country, source, and time window
  • Drag nodes to inspect clusters. Use the zoom and reset controls

Event types and colors:

const EVENT_TYPES = {

  Humanitarian:{ color:"#8affc1" },

Country centroids and regions live in COUNTRY_INDEX. Add your own entries as shown below.

"Somaliland": { c:[9.55, 44.05], r:"Africa", iso:"SO" }

Network link heuristics:

// link weight
if (a.type === b.type) w += 1;
if (a.country && b.country && a.country === b.country) w += 1;
if (a.source === b.source) w += 0.8;
if (Math.abs(a.date - b.date) < 7*24*60*60*1000) w += 0.6;
if (w >= 1.4) addLink();

Deploy

  • Static hosts like GitHub Pages, Netlify, Cloudflare Pages, S3 are supported
  • Host your own proxy for reliability and to control CORS
  • Respect tile provider attribution if you swap map layers

Troubleshoot

  • Blank map or no data - check the browser console. If CORS errors appear, set your own proxy URL
  • Feeds stall - AllOrigins can rate limit. Reduce feed list or switch to your worker
  • Markers missing - coordinates or country match not found. Add to COUNTRY_INDEX or enrich feeds server side
  • Slow UI - lower SOFT_LIMIT_ITEMS and reduce heatmap radius
